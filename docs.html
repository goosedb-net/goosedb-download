Getting Started

GooseDB는 DuckDB를 기반으로 한 확장 데이터베이스 시스템으로, PostgreSQL 와이어 프로토콜을 지원하여 기존 PostgreSQL 도구 및 클라이언트와 완벽하게 호환됩니다.

Note: GooseDB를 시작하기 전에 시스템 요구사항을 확인하시기 바랍니다.

Installation

Using pip

$ pip install goosedb
