Installation Guide

Using Package Manager

# Using pip
$ pip install goosedb

# Using Homebrew
$ brew install goosedb

Manual Installation

$ wget https://goosedb.org/releases/1.0.0/goosedb.tar.gz
$ tar xzf goosedb.tar.gz
$ cd goosedb
$ ./install.sh
